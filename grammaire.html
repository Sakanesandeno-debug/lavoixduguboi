Grammaire · Langue Guboï

Premiers éléments de grammaire du Guboï

La langue Guboï est la langue du peuple Iboï, originaire de la terre de Kaboï. Traditionnellement orale, elle commence aujourd'hui à être documentée afin de préserver sa richesse linguistique et culturelle. Cette page présente quelques éléments fondamentaux de la structure du Guboï.

1.

L'infinitif

En Guboï, l'infinitif se forme avec la particule Be placée devant la base verbale.

Structure : Be + radical verbal
  • Be zup : parler
  • Be gnam : manger (un repas)
  • Be hob : croquer, manger un fruit
  • Be yaab : boire
  • Be hem : dormir
  • Be fa : partir
2.

Les pronoms personnels sujets

3.

L'aspect habituel

Le présent peut exprimer une habitude, une capacité ou une vérité générale.

Exemple :
Me ma zupa Guboï
Je parle Guboï / Je suis locuteur du Guboï.
4.

L'aspect progressif

Le progressif exprime une action en cours.

Structure : Sujet + ngo + verbe progressif
Exemple :
Ma ngo agnamta
Je suis en train de manger.
À la troisième personne, le n peut disparaître : Â go agnamta — Il/elle est en train de manger.
5.

L'aspect perfectif

Le perfectif exprime une action accomplie.

  • Ma gnami — J'ai mangé (un repas).
  • Ma kobi — J'ai croqué/mangé quelque chose.
6.

Un verbe irrégulier : Be hob

Be hob signifie croquer ou manger un fruit.

7 .

Le singulier et le pluriel des noms

En Guboï, le pluriel des noms se forme par un changement de préfixe. Ces préfixes varient selon la catégorie du nom : être humain, végétal ou chose. On peut identifier plusieurs logiques dans ces accords.

7.1

Les noms commençant par A-

Les noms commençant par le préfixe A- font leur pluriel avec le préfixe GA-.

STRUCTURE : A- (singulier) → GA- (pluriel)
SingulierPlurielSens
A-mangoGa-mangola mangue / les mangues
A-buGa-bule ventre / les ventres
A-bahGa-bahla jambe / les jambes
A-doGa-lole tubercule d'igname / les tubercules (irrégulier)
A-dekoGa-lekole tubercule de manioc / les tubercules
A-demnaGa-lemnale citron / les citrons (irrégulier)
A-roGe-rol'animal / les animaux (légère variation)
7.2

Les noms commençant par U- désignant un végétal

Les noms commençant par U- et désignant un végétal font leur pluriel avec le préfixe DA- ou DE-.

STRUCTURE : U- (singulier) → DA- / DE- (pluriel)
SingulierPlurielSens
U-mangoDa-mangole manguier / les manguiers
U-ndemnaDa-ndemnale citronnier / les citronniers
U-ndenoDa-ndenole kapokier / les kapokiers
U-doDe-dol'arbre / les arbres (légère variation)
7.3

Les noms commençant par U- désignant un être humain

Les noms commençant par U- et désignant une personne font leur pluriel avec le préfixe I-.

STRUCTURE : U- (singulier) → I- (pluriel)
SingulierPlurielSens
U'liI-lila personne / les personnes
U-ligenI-ligenmonsieur / les messieurs
U-likamI-likamla dame / les dames
ATTENTION :

Le genre se marque avec le préfixe DJA- : U-lika → Dja-likam (féminin) / U-ligé → Dja-ligen (masculin)

Ces exemples montrent que le Guboï possède une structure verbale riche, avec des alternances de radical et une distinction claire entre différents aspects de l'action. Cette documentation constitue une première contribution à la description et à la préservation de la langue Guboï.
